SSH Access:

  • ssh -p ${ctx.ssh_access.port} odoo@${ctx.ssh_access.domain}
  • ssh -p ${ctx.ssh_access.port} root@${ctx.ssh_access.domain}

To verify the odoo logs see:

  • /var/log/supervisor/odoo_stdout.log

To verify the postgres logs see:

  • /var/log/supervisor/postgres_stdout.log
  • /var/log/supervisor/postgres_stderr.log

To manage the instance of odoo use supervisorctl:

  • supervisorctl start odoo
  • supervisorctl stop odoo
  • supervisorctl status odoo
  • supervisorctl restart odoo

Warning!

You must not stop the supervisor process!
